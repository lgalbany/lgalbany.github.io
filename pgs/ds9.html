Follow these instructions to add some histogram functionality to the ds9 viewer. Once you have this stuff property installed you can draw a region in ds9 then do one of four things with the pixel data inside the region:

1. dump the raw pixel values

2. make an ascii-based historgram

3. make a graphical histogram (using gunplot)

4. make a graphical histogram (using paw)

These instructions assume that you will be running ds9 as user monsoon.

0. make sure you've got enough disk space on the "/" parition: df -h

1. install funtools (in /usr/local/bin)

1a. download the latest funtools

1b. gunzip the file: gunzip filename.tar.gz

1c. untar the tarball: tar -xf

1d. cd into the directory that was just created

1e. ./configure

1f. make

1g. (as root) make install

2. add /usr/local/bin to the path (edit /home/monsoon/.cshrc)

3. (as root) mkdir /usr/tools

4. (as root) copy tools.zip to /usr/tools

4b. (as root) unzip tools.zip

5. log out then log back in

6. open a fits file with ds9, draw a region

7. look under the analysis menu. at the bottom there should be four new items for analyzing the pixel data in the region you just defined.
